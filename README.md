## Overview
A comprehensive platform for booking convention centers and wedding halls in India. The system connects customers, hall owners, and administrators through dedicated interfaces.

## Deliverables
1.  **Android App** (Customer)
2.  **iOS App** (Customer)
3.  **Web Admin Panel** (Admin)
4.  **Web Hall Owner Dashboard** (Owner)

## Key Features
-   **Market**: India Only (IN)
-   **Payments**: UPI Integrated
-   **Design**: Wedding-themed (Royal Blue + Gold)
-   **Tech Stack**:
    -   Frontend: React (Web/Prototype), React Native (Mobile - Planned)
    -   Backend: Node.js + Express
    -   Database: PostgreSQL
